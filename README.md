# cns-wc26cup-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面彼此独立，不隶属于任何特定网站或域名，仅作为静态内容存档与展示使用。

## 项目简介

cns-wc26cup-lab 是一个轻量级的 HTML 页面归档仓库。仓库内的每个页面均为独立构建，不依赖外部框架或后端服务。适用于存放各类实验性、展示性或临时性的 HTML 内容。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（CSS、JS、图片等，如有）
    ├── style.css
    └── ...
```

- `pages/`：所有独立 HTML 页面存放于此。每个页面保持完整，可单独访问。
- `assets/`：页面共享的静态资源（如需统一样式或脚本）。

## 页面归档说明

- 每个页面均为自包含的 HTML 文件，可直接在浏览器中打开。
- 页面之间无相互依赖关系，可独立更新或删除。
- 仓库不提供后端接口或动态数据支持，所有内容均为静态。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面或改进现有页面。
- 提交前请确保页面格式正确，无外部恶意引用。
- 如需删除或更新页面，请直接修改 `pages/` 目录下的对应文件。
- 本仓库不承担页面内容的长期可用性保证，请自行备份重要内容。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/cns-wc26cup-lab.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署整个仓库。
